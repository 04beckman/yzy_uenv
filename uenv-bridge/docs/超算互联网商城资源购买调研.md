调研对象：

- 活动页：https://www.scnet.cn/ui/acm/index.html#/activity?invitation_code=18221793699&isNew=1&activityName=v47N2Wvl
- 算力市场：https://www.scnet.cn/ui/mall/search/resource
- 应用商城：https://www.scnet.cn/ui/mall/search/goods

调研时间：2026-07-01

## 结论

该网站不只提供活动页里的 3 个共享试用资源，还可以通过“算力市场”和“应用商城”购买或获取其他资源。

活动页本身仍然是试用申请入口：只看到 3 个共享试用资源，不展示正式购买价格，也没有独占或裸金属切换入口。正式购买入口在商城侧，尤其是 `/ui/mall/search/resource` 的算力市场。

公开接口显示，算力市场当前可检索到 74 条基础资源，覆盖共享型、独占型、裸金属、存储等交付形态；普通应用商城当前可检索到 6890 条商品记录，按商品 ID 去重后约 6065 个商品，覆盖应用平台、应用软件、技术服务、模型、数据、源代码等类别。

## 数据来源与口径

公开列表接口：

- 算力资源列表：`POST https://www.scnet.cn/acx/search/quantumCompute/full-text`
- 普通商品列表：`POST https://www.scnet.cn/acx/search/goods/v1/full-text`
- 商品分类树：`GET https://www.scnet.cn/acx/mall/goods-type/category`
- 站点配置：`GET https://www.scnet.cn/acx/resource/config/index`

调研使用未登录状态下的公开接口，因此能确认公开展示的商品分类、标价、计费单位和购买链路，但不能确认登录后用户级折扣、团队权限、库存锁定、最终订单金额和支付资格。

下面按具体 SKU 列出公开价格。`公开价格` 为列表接口返回的 `basePrice`，最终成交价仍需登录后在购物车和订单页确认。

表中的“后端资源类型字段”来自算力资源列表接口返回的 `extra.goodsResourceType`，例如 `DCU`、`GPU`、`CPU`、`DCUMetal`。网站资源卡可见文案通常只写“异构加速卡1 显存16GB”“异构计算”等，不直接展示 `DCU`。因此这里的 `DCU` 应理解为后端商品分类字段，不是页面明示的硬件品牌或型号；如果只按网页可见内容，应表述为“异构加速卡1/2”。

`DCUMetal` 可按后端枚举拆成 `DCU` + `Metal` 理解：`DCU` 表示该商品归在 DCU/异构加速卡资源类型下，`Metal` 表示裸金属形态。它对应页面标签里的“独占型、裸金属、异构型”，不是页面展示的具体硬件型号。

服务形态判断来自商品字段和页面标签：

- `extra.goodsType=SHARE`，页面标签为“共享型”，按 `核*时` 或 `卡*时` 计价。
- `extra.goodsType=CONTRACT`，页面标签为“独占型”，通用/异构资源多按 `台*月` 或 `台*天` 计价。
- `extra.resourceType=BMS`，页面标签包含“裸金属”，且同时标记为“独占型”，按 `台*天` 计价。
- `extra.goodsType=STORAGE`，页面标签为“存储”，按 `GB*月` 计价。

很多普通商品公开列表里 `basePrice=0` 或没有 `priceUnit`。这不一定都表示完全免费，可能是免费获取、使用已有算力、联系商家、登录后选择规格，或详情页/订单页再确认价格。因此普通商品的最终价格应以登录后的商品规格、购物车和订单确认页为准。

## 购买链路

前端路由和接口显示，商城具备完整购买链路：

1. 浏览列表：`/ui/mall/search/resource` 或 `/ui/mall/search/goods`。
2. 进入商品详情：资源卡点击后进入商品详情页，普通商品也进入商品详情页。
3. 加入购物车或立即购买：购物车接口为 `/acx/market/isc/shoppingcart`，支持 `GET/POST/PUT/DELETE`。
4. 购物车：前端路由 `/goods/cart`。
5. 订单确认：前端路由 `/goods/order`。
6. 支付：前端路由 `/goods/pay`，并存在 `/goods/pay/charge`、`/goods/pay/scan`、`/goods/pay/remittance-to-platform`、`/goods/pay/remittance-to-merchants`、`/goods/pay/result` 等支付/汇款结果页面。

购物车前端逻辑体现的计价方式：

- 共享算力：按单价乘购买数量，单位是 `核*时` 或 `卡*时`。
- 独占算力：按单价乘数量和购买时长，单位是 `台*月` 或 `台*天`。
- 存储资源：按容量和时长计价，单位是 `GB*月`。
- 普通商品：按商品自己的 `billingType` 和价格单位展示，常见单位是 `次`、`月`、`年`、`天`。

站点配置接口返回 `purchaseEnable=true`，说明商城购买能力处于开启状态。但实际下单、支付、团队选择、实名认证、库存校验和发票/合同等流程需要登录后才能完成。

## 与活动页的区别

| 维度 | 活动页 | 商城 |
| --- | --- | --- |
| 入口 | `/ui/acm/index.html#/activity...` | `/ui/mall/search/resource`、`/ui/mall/search/goods` |
| 目的 | 邀请码试用申请 | 正式商品浏览、购买、试用或获取 |
| 当前可见资源 | 3 个共享试用资源 | 74 条算力资源，6065 个去重普通商品 |
| 价格展示 | 不展示人民币单价 | 展示 `basePrice` 和 `priceUnit`，部分商品价格为 0 或未展示单位 |
| 共享 | 有，仅共享试用 | 有，共享型算力按核时/卡时 |
| 独占 | 活动页未发现 | 有，独占型算力按台月/台天 |
| 裸金属 | 活动页未发现 | 有，BMS 裸金属按台天 |
| 存储 | 活动页只有申请字段 | 有，存储按 GB*月 |
| 订单/支付 | 无正式支付链路 | 有购物车、订单、支付、汇款页面 |

## 结论性判断

可以购买其他资源。对于算力资源，网站当前公开可见的购买形态包括共享型、独占型、裸金属和存储；计费单位分别以 `核*时`、`卡*时`、`台*月`、`台*天`、`GB*月` 为主。

对于非算力资源，网站还有应用平台、应用软件、技术服务、模型、数据、源代码等普通商城商品，其中技术服务、应用平台、应用软件和少量数据商品能看到正价；模型和源代码类商品在公开列表中大多以 0 元或未展示单位出现，需要进入详情和登录后的订单页确认最终获取方式。

如果要从该网站采购正式算力，应优先走“算力市场”，因为那里明确区分共享、独占、裸金属和存储资源，并直接展示算力计费单位；活动页只适合领取试用额度，不能代表正式购买能力。
