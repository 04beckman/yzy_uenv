## 实施备注

1. **代码同步**：Windows tar 打包 → scp → 解压（`gen/` 单独同步，因 gitignore）
2. **编译修复**：`uenv-worker/src/plugin/arpc/mod.rs` 需 `TokioIo` 包装 UDS（tonic 0.14）
3. **grpcurl**：`payload`/`reward_config` 为 bytes，须 base64 编码
4. **CLI 顺序**：`uenv-worker --config <yaml> serve`（非 `serve --config`）
5. **插件**：须设置 `UENV_GSM8K_PLUGIN_BIN=/root/UEnv/target/release/uenv-gsm8k-plugin`
